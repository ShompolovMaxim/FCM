## Общее описание

В данном репозитории размещён код "Программы для моделирования результатов обсуждения проблемных ситуаций на основе нечетких когнитивных карт", реализованной в рамках выполнения Выпускной квалификационной работы в рамках обучения на образовательной программе "Программная инженерия" НИУ ВШЭ.

"Программа для моделирования результатов обсуждения проблемных ситуаций на основе нечетких когнитивных карт" - десктопное приложение, позволяющее строить нечёткие когнитивные карты на основе отдельных или агрегированных экспертных оценок, а также осуществлять симуляцию и прогнозирование результатов обсуждения проблемной ситуации, исходя из заданной модели. Программа должна представлять из себя инструмент, позволяющий моделировать предметную область при помощи нечётких когнитивных карт и производить пошаговый расчёт значений факторов, а также создавать основу для формулирования выводов для проводимой дискуссии по проблемной ситуации.

## Технологии реализации

Язык программирования: C++

Фреймворк для построения графического интерфейса:
- Qt6.8.4
- Релиз: 4.02.2025
- LTS до 8.10.2029
- Официальный сайт: https://www.qt.io
- Разработчик: The Qt Company

Библиотека для построения графиков:
- QCustomPlot 2.1.1
- Официальный сайт: https://www.qcustomplot.com
- Автор: Эйке Хеттхер (Eike Heitter)

Библиотека для матричных вычислений: Eigen 5.0.0

Библиотека для юнит-текстирования: gtest

СУБД: SQLite 3

## Структура репозитория

- docs – директория с программной документацией
- src – директория с исходным кодом проекта:
    - libs – директория с библиотеками:
        - eigen – директория с библиотекой матричных вычислений Eigen 5.0.0 (ссылка на репозиторий)
        - googletest – директория с библиотекой юнит-тестирования googletest 1.17.0 (ссылка на репозиторий)
        - qcustomplot – директория с библиотекой построения визуализаций QCustomPlot 2.1.1 (исходный код)
    - migrations – директория с миграциями базы данных
    - model – директория слоя модель (содержит сущности и логику работы с нечёткими когнитивными картами, не зависящую от пользовательского интерфейса):
        - activation_functions – директория с функциями активации
        - algorithms – директория с алгоритмами прогнозирования
        - entities – директория с сущностями
        - fuzzy_logic – директория с преобразованиями нечёткой логики
        - metrics – директория с метриками
        - static_analysis – директория с логикой статического анализа НКК
        - stop_conditions – директория с условиями остановки прогнозирования
    - presenter – директория, содержащая презентеры
    - repository – директория с модулями доступа к данным
    - ui – директория со структурой и логикой пользовательского интерфейса (представлением):
        - concept_window – директория окна фактора
        - graph_editor – директория интерактивной области редактирования графа НКК
        - load_model_window – директория окна загрузки модели
        - main_window – директория основного окна
        - weight_window - директория окна связи
    - resources.qrc – файл ресурсов Qt
- translations – директория с файлами перевода с английского языка
- CMakeLists.txt – файл сборки
